# Practica 3.3 - Juego Pygame

## Requisitos

- Python 3.10+
- Pygame

## Instalacion

```bash
pip install -r requirements.txt
```

## Ejecucion

```bash
python main.py
```

## Controles

- Movimiento: W A S D o flechas
- Disparo: Espacio
- Reiniciar (al terminar): R
- Salir (al terminar): ESC

## Nota sobre fuentes

El juego intenta cargar una fuente importada desde:

- `assets/fonts/PressStart2P-Regular.ttf`

Si no existe, usa una fuente del sistema para que el juego siga funcionando.
